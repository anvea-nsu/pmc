# Прием и передача данных

## Задание

Осуществить прием командной строки. Формат строки: номер устройства, функция (запись/чтение), адрес регистра, данные. Выполнить разбор команды. Выполнить команду (если функция чтение, то прочитать содержимое регистра в памяти, если функция запись, то обновить значение регистра в памяти). При выбранной функции чтения выполнить обратную передачу сообщения по формату: номер устройства, адрес регистра, данные. Продемонстрировать работу с использованием SimulIDE или реального устройства и терминальной программы.
