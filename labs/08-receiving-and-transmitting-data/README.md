# Прием и передача данных

## Задание

Осуществить прием и обратную передачу строки (набор из 10 символов, последний из которых знак конца строки). Принятая строка записывается в память микроконтроллера. Конфигурация приемо-передатчика (UART) должна быть настроена на использование прерываний. Продемонстрировать работу с использованием SimulIDE или реального устройства и терминальной программы.
