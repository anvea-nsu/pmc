# Аналого-цифровой преобразователь

## Задание 

Настроить АЦП на однократное преобразование значения сигнала на одном из входов модуля. Продемонстрировать работу с использованием SimulIDE или реального устройства и терминальной программы. При демонстрации на реальном устройстве необходимо дополнительно настроить UART и осуществить отправку кода АЦП.
