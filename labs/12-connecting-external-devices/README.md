# Подключение внешних устройств.

Осуществить считывание состояния кнопочной клавиатуры, подключенных к входному порту микроконтроллера. Выполнить отображение цифры на семисегментном индикаторе соответствующей номеру нажатой кнопки. Продемонстрировать работу с использованием SimulIDE или реального устройства.
