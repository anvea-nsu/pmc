# EEPROM.

Считать байт по UART (используется предыдущая работа), записать байт в память EEPROM, прочитать этот же байт из EEPROM, добавить единицу и отправить обратно на терминал через UART. Ожидать приема нового байта. Продемонстрировать работу с использованием SimulIDE или реального устройства и терминальной программы.
