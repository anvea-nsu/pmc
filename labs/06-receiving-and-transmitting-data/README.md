# Приём и передача данных. UART на ATmega168A

## Задание

Настроить приемо-передатчик (UART) для приема и передачи одного байта. Продемонстрировать прием и отправку байта в программе эмуляторе. Продемонстрировать работу с использованием SimulIDE или реального устройства и терминальной программы.

## Параметры UART

- Скорость: 9600 бод
- Формат: 8N1 (8 бит данных, без чётности, 1 стоп-бит)
- Тактовая частота МК: 16 МГц

## Сборка проекта

1. Открыть `main.asm` в **Microchip Studio**
2. **Build → Build Solution** (`F7`)
3. В папке `Debug/` внутри папки проекта появится файл `asm_project.hex`

## Запуск симуляции в SimulIDE

1. Открыть **SimulIDE**
2. **Open Circuit** → выбрать файл `circuit.sim1` из этого репозитория
3. Правой кнопкой мыши на плату ATmega168 → **Load Firmware** → выбрать `asm_project.hex`
4. Нажать **Start Simulation**
5. Нажать кнопку **Open** на компоненте Serial Terminal
6. Написать любое сообщение в поле **Send Text** и отправить

**Ожидаемый результат:** срабатывает индикатор TX терминала (отправка в МК), затем RX (ответ от МК) — и то же сообщение возвращается обратно в поле Input.
